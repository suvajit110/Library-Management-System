# Library Management System

A Java Swing based Library Management System developed using NetBeans IDE and MySQL database.
This application helps in managing books, students, issuing and returning books, and maintaining records
with secure login and logout functionality.

## Features
- Admin Login & Logout
- Manage Books
- Manage Students
- Issue & Return Books
- View Issued Books and Records

## Technologies Used
- Java (Swing)
- MySQL
- JDBC
- NetBeans IDE
